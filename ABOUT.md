# About This Workshop

## Overview

This is a hands-on workshop for learning Cloudflare Workers, R2 storage, D1 database, and Workers AI. It is designed for beginners with no prior experience with Cloudflare.

## What You Will Build

By the end of this workshop, you will have built:

1. **Hello World Worker** - Learn the basics of Cloudflare Workers
2. **Personal Profile Page** - A Linktree-style profile page
3. **Image Gallery** - Store and serve images with R2
4. **Instagram-Style Photo App** - Upload photos with captions using R2 + D1
5. **AI Chatbot** - Chat with multiple AI models using Workers AI
6. **Complete App** - A full application combining AI chat and media gallery

## Technologies Used

| Technology | Description |
|------------|-------------|
| Cloudflare Workers | Serverless functions running on the edge |
| Cloudflare R2 | Object storage for files (images, videos) |
| Cloudflare D1 | SQLite database at the edge |
| Workers AI | AI models running on Cloudflare |
| Wrangler CLI | Command-line tool for Cloudflare development |

## Target Audience

- Beginners learning web development
- Developers new to Cloudflare
- Anyone interested in serverless and edge computing

## Prerequisites

- Basic understanding of HTML, CSS, and JavaScript
- A computer with Windows or macOS
- Internet connection
- Free Cloudflare account

## Workshop Duration

Approximately 2.5 hours to complete all modules.
